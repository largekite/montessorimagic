# MontessoriMagic

> Personalized Montessori learning adventures for every child

MontessoriMagic creates AI-powered, adaptive learning books tailored to your child's age, interests, and developmental stage. Parents track progress and watch their kids fall in love with learning through beautifully crafted Montessori-inspired stories and activities.

Generated by **[Skylia](https://skylia.dev)** AI Startup Builder.

## Stack

| Layer | Service |
|-------|---------|
| Framework | Next.js 14 + TypeScript |
| Styling | Tailwind CSS + shadcn/ui |
| Hosting | Vercel |
| Database | Neon + Prisma |
| Auth | Clerk |
| Payments | Stripe |
| Email | Resend |
| AI | OpenAI |


## Getting started

```bash
npm install
cp .env.example .env.local
# Fill in your environment variables
npm run dev
```

### Where to get them

- `DATABASE_URL` — Neon PostgreSQL connection string (Neon dashboard)
- `NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY` — Clerk publishable key for client-side auth (Clerk dashboard)
- `CLERK_SECRET_KEY` — Clerk secret key for server-side auth (Clerk dashboard)
- `CLERK_WEBHOOK_SECRET` — Clerk webhook signing secret for user sync (Clerk dashboard)
- `STRIPE_SECRET_KEY` — Stripe secret key for payment processing (Stripe dashboard)
- `NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY` — Stripe publishable key for client-side Stripe.js (Stripe dashboard)
- `STRIPE_WEBHOOK_SECRET` — Stripe webhook signing secret (Stripe dashboard)
- `STRIPE_FAMILY_PRICE_ID` — Stripe price ID for the Family plan (Stripe dashboard)
- `STRIPE_UNLIMITED_PRICE_ID` — Stripe price ID for the Unlimited plan (Stripe dashboard)
- `RESEND_API_KEY` — Resend API key for transactional emails (Resend dashboard)
- `RESEND_FROM_EMAIL` — Verified sender email address for Resend (Resend dashboard)
- `OPENAI_API_KEY` — OpenAI API key for book generation and activity suggestions (OpenAI dashboard)
- `NEXT_PUBLIC_POSTHOG_KEY` — PostHog project API key for analytics (PostHog dashboard)
- `NEXT_PUBLIC_POSTHOG_HOST` — PostHog instance host URL (PostHog dashboard)
- `SENTRY_DSN` — Sentry DSN for error monitoring (Sentry dashboard)
- `NEXT_PUBLIC_APP_URL` — Public base URL of the deployed application (Vercel environment settings)

## Scripts

- `npm run dev` — start development server

- `npm run db:push` — push schema to database (dev)
- `npm run db:migrate` — run migrations (production)
- `npm run db:studio` — open Prisma Studio
